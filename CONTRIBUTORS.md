# 기여자 명단 

이 수업에 참여한 모든 분들이 여기에 이름을 남깁니다.
아래 방법대로 따라하면 여러분의 이름이 이 저장소에 영원히 남아요!

---

## 참여 방법

### 1단계 — Fork
이 페이지 오른쪽 상단 **Fork** 버튼을 클릭하세요.
내 GitHub 계정으로 이 저장소 전체가 복사됩니다.

> Fork가 완료되면 주소가 `github.com/내아이디/glp2` 로 바뀌어 있을 거예요.

---

### 2단계 — 내 파일 만들기
내 Fork 저장소에서:

1. **`Add file`** 버튼 클릭 → **`Create new file`** 클릭
2. 파일명 입력란에 아래처럼 입력하세요:
   ```
   contributors/김북곽.md
   ```
   - `contributors/` 를 앞에 붙이면 자동으로 폴더가 생겨요!
   - `김북곽` 부분을 **본인 이름**으로 바꿔주세요

3. 본문에 아래 내용을 입력하세요:
   ```markdown
   # 김북곽
   
   한마디: "오픈소스의 거인이 될 거예요!"
   ```
   - `김북곽`과 한마디는 자유롭게 바꾸셔도 돼요 

---

### 3단계 — Commit
오른쪽 위 **`Commit changes`** 버튼 클릭
→ 커밋 메시지는 그대로 두거나 원하는 내용으로 바꾸기
→ **`Commit changes`** 클릭

---

### 4단계 — Pull Request 보내기
1. 상단에 **`Contribute`** 버튼 → **`Open pull request`** 클릭
2. 제목 확인 후 **`Create pull request`** 클릭 

---

### 5단계 — contributors 폴더에서 내 이름 확인하기! 
선생님이 PR을 확인하고 merge해주면
contributors 폴더에 내 이름이 추가됩니다!



---

> **왜 파일을 따로 만드나요?**
> 같은 파일을 여러 명이 동시에 수정하면 충돌(Conflict)이 생겨요.
> 각자 다른 파일을 만들면 충돌 없이 모두의 PR이 merge됩니다!
